# Release Notes

## v1.0
Реализован краулер:

- Скачивание HTML-страниц по заранее подготовленному списку URL (`data/urls.txt`)
- Сохранение каждой страницы в отдельный файл в `output/pages/` 
- Формирование `output/index.txt` 
- Обработка ошибок загрузки 
- Упаковка результата в архив `output/pages.zip`

## v2.0

Реализована токенизация и лемматизация:
- Извлечение уникальных токенов из скачанных HTML-страниц
- Фильтрация союзов, предлогов, чисел и непонятных токенов (слова с одновременным наличием букв и цифр, обрывки разметки)
- Группировка токенов по леммам с помощью `pymorphy2`
- Сохранение результатов:
  - `output/text/tokens.txt` — токены, по одному на строку
  - `output/text/lemmas.txt` — строки вида `<лемма> <токен1> <токен2> ...`

## v3.0

Реализован инвертированный индекс и булев поиск:
- Построение инвертированного индекса по леммам страниц (`build-inverted`)
- Сохранение индекса в `output/inverted_index.txt`
- Булев поиск по строковому запросу (`boolean-search`)
- Поддержка операторов `AND`, `OR`, `NOT`
- Поддержка сложных запросов со скобками
- Возможность передавать запрос через CLI-параметр `--query` (без хардкода)
